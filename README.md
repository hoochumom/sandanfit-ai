# 산단핏 AI

산업단지 입주기업을 위한 AI 성장진단 PC 웹 MVP입니다.

## 구성

- `index.html`: 화면 구조
- `styles.css`: PC 웹 대시보드 스타일
- `app.js`: 기업 진단, 협력사 선택, 리포트 버튼 등 데모 상호작용

## 로컬에서 보기

저장소를 다운로드하거나 clone한 뒤 `index.html`을 브라우저로 열면 됩니다.

```bash
python -m http.server 4186
```

서버를 띄운 경우 브라우저에서 아래 주소로 접속합니다.

```text
http://127.0.0.1:4186/
```
